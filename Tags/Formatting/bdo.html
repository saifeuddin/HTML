## HTML <bdo> Tag


BDI stands for Bi-Directional Override.

$$ -- Used to override the current text direction.



Attributes :

  • dir
    • ltr
    • rtl

Global Attributes

$$ -- Supports the Global Attributes in HTML.

Event Attributes

$$ -- Supports the Event Attributes in HTML.

## HTML <cite> Tag


$$ -- Defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.).

Note :

A person's name is not the title of a work.

The text in the cite element usually renders in italic.

Global Attributes

$$ -- Supports the Global Attributes in HTML.

Event Attributes

$$ -- Supports the Event Attributes in HTML.


Example :

The Scream by Edward Munch. Painted in 1893.

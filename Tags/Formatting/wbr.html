## HTML <wbr> Tag


$$ -- The (Word Break Opportunity) tag specifies where in a text it would be ok to add a line-break.

Tip:
When a word is too long, the browser might break it at the wrong place. You can use the element to add word break opportunities.

$$ -- Supports the Global Attributes in HTML.

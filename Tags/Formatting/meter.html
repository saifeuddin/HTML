## HTML <meter> Tag


$$ -- Defines a scalar measurement within a known range, or a fractional value. This is also known as a gauge.

Examples: Disk usage, the relevance of a query result, etc.

Tip :

Always add the <label> tag for best accessibility practices!



Note :

Note: The meter tag should not be used to indicate progress (as in a progress bar). For progress bars, use the progress tag.



Attributes :

  1. form = form-id

    Specifies which form the meter element belongs to.

  2. id = id name

    Used with label tag.

  3. name = name

  4. min = number

    Specifies the minimum value of the range. Default value is 0.

  5. max = number

    Specifies the maximum value of the range.

  6. low = number

    Specifies the range that is considered to be a low value.

  7. high = number

    Specifies the range that is considered to be a high value

  8. optimum = number

    Specifies what value is the optimal value for the gauge

  9. value = number

    Required. Specifies the current value of the gauge.

Global Attributes

$$ -- Supports the Global Attributes in HTML.

Event Attributes

$$ -- Supports the Event Attributes in HTML.

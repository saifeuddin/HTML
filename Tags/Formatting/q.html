## HTML <q> Tag


$$ -- defines a short quotation.

Browsers normally insert quotation marks around the quotation.



Attributes :

  • cite = URL

Global Attributes

$$ -- Supports the Global Attributes in HTML.

Event Attributes

$$ -- Supports the Event Attributes in HTML.

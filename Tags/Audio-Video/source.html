## HTML <source> Tag





All Attributes

  1. media - media_query

    Accepts any valid media query that would normally be defined in a CSS. 

                    
                    
    <picture>  
        <source media="(min-width:650)" srcset="img_pink_flowers.jpg">
        <source media="(min-width:450)" srcset="img_white_flowers.jpg"> 

        <img src="img_oragnge_flowers.jpg" alt="flowers" style="width:auto;"> 
    </picture>

  2. src - URL

    Required when <source> is used in <audio> and <video>. Specifies the URL of the media file. 

                    
                
    <video muted autoplay loop >
        <source src="movie.mp4" type="video/mp4" > 
            Your browser does not support the video element.
    </video>

  3. srcset - URL

    Required when <source> is used in <picture>. Specifies the URL of the image to use in different situations 

                    
                
    <picture>  
        <source srcset="img_pink_flowers.jpg">
        <source srcset="img_white_flowers.jpg"> 

        <img src="img_oragnge_flowers.jpg" alt="flowers" style="width:auto;"> 
    </picture>

  4. type - MIME-type

    Specifies the MIME-type of the resource 

                    
                
    <video muted autoplay loop >
        <source src="movie.mp4" type="video/mp4" > 
            Your browser does not support the video element.
    </video>

  5. sizes

    Specifies image sizes for different page layouts

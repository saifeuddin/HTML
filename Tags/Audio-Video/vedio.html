## HTML <video> Tag





All Attributes

  1. height

    Sets the height of the video player. 

                    
                    
    <video height= "300px">  </video>

  2. width

    Sets the width of the video player 

                    
                    
    <video height= "300" width="300" > </video>

  3. poster - URL

    Specifies an image to be shown while the video is downloading, or until the user hits the play button. 

                    
                    
    <video height= "300" width="300" poster="images/w3html5.gif"> 
    
    </video>

  4. autoplay

    Specifies that the video will start playing as soon as it is ready. 

                    
                    
    <video autoplay>  </video>


  5. controls

    Specifies that video controls should be displayed (such as a play/pause button etc). 

                    
                
    <video controls autoplay>  </video>

  6. loop

    Specifies that the video will start over again, every time it is finished. 

                    
                
    <video autoplay loop >  </video>

  7. muted

    Specifies that the audio output of the video should be muted. 

                    
                
    <video muted autoplay loop >  </video>

  8. src - URL

    Specifies the URL of the video file. 

                    
                
    <video src="movie.mp4" muted autoplay loop >
        Your browser does not support the video element.
    </video>

  9. preload - auto | none | metadata

    The preload attribute specifies if and how the author thinks that the video file should be loaded when the page loads. 

                    
                
    <video controls preload="none">
        
    </video>

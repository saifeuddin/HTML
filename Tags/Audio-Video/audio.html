## HTML <audio> Tag





All Attributes

  1. autoplay

    Specifies that the audio will start playing as soon as it is ready. 

                    
                    
    <audio autoplay>  </audio>


  2. controls

    Specifies a list of image files to use in different situations. 

                    
                
    <audio controls autoplay>  </audio>

  3. loop

    Specifies that the audio will start over again, every time it is finished. 

                    
                
    <audio autoplay loop >  </audio>

  4. muted

    Specifies that the audio output should be muted. 

                    
                
    <audio muted autoplay loop >  </audio>

  5. src - URL

    Specifies the URL of the audio file. 

                    
                
    <audio src="horse.ogg" muted autoplay loop >
        Your browser does not support the audio element.
    </audio>

  6. preload

    The preload attribute specifies if and how the author thinks that the audio file should be loaded when the page loads. 

                    
                
    <audio controls preload="none">
        
    </audio>

## HTML <frame> Tag referrerpolicy Attribute


$$ -- The referrerpolicy attribute specifies which referrer information to use when fetching an image.




Syntax

        
        
    <iframe referrerpolicy=" no-referrer| no-referrer-when-downgrade | 
        origin|origin-when-cross-origin|same-origin|strict-origin-when-cross-origin | unsafe-url"">

Attributes Values

        
        
    <iframe src="demo_iframe_sandbox.htm" sandbox referrerpolicy="no-referrer">

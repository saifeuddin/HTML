## HTML <frame> Tag sandbox Attribute


$$ -- The sandbox attribute enables an extra set of restrictions for the content in the iframe.



When the sandbox attribute is present and it will:

The value of the sandbox attribute can either be empty (then all restrictions are applied), or a space-separated list of pre-defined values that will REMOVE the particular restrictions.




Syntax

        
        
    <iframe sandbox="value">

Attributes Values

        
        
    <iframe src="demo_iframe_sandbox.htm" sandbox>
    <iframe src="demo_iframe_sandbox.htm" sandbox="allow-scripts">
    <iframe src="demo_iframe_sandbox.htm" sandbox="allow-forms">

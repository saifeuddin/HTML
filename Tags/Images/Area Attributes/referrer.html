## HTML <area> Tag referrerpolicy Attribute


$$ -- Specifies which referrer information to send with the link.




Syntax

        
        
    <area referrerpolicy=" no-referrer| no-referrer-when-downgrade | 
        origin|origin-when-cross-origin|same-origin|strict-origin-when-cross-origin | unsafe-url"">

Attributes Values

        
        
    <area shape="circle" coords="124,58,8" href="venus.htm" alt="Venus" referrerpolicy="same-origin">

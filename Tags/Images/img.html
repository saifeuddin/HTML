## HTML <img> Tag


$$ -- The img tag is used to embed an image in an HTML page.


The img tag has two required attributes:
  • src - Specifies the path to the image.
  • alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed.


Tip: To link an image to another document, simply nest the img tag inside an <a> tag (see example below).


All Attributes

  1. src - URL

    Specifies the path to the image 

                    
        <img src="default.jpg">
  2. srcset - URL-list

    Specifies a list of image files to use in different situations.

  3. alt - text

    Specifies an alternate text for the image, if the image for some reason cannot be displayed.

  4. longdesc - URL

    Specifies a URL to a detailed description of an image.



  5. height - pixels


  6. width - pixels


  7. sizes - sizes

    Specifies image sizes for different page layouts



  8. usemap - value

    Specifies an image as a client-side image map

  9. ismap

    Specifies an image as a server-side image map.



  10. crossorigin = anonymous / use-credentials

    Allow images from third-party sites that allow cross-origin access to be used with canvas.


  11. loading = eager / lazy

    Specifies whether a browser should load an image immediately or to defer loading of images until some conditions are met.


  12. referrerpolicy = values

    Specifies which referrer information to use when fetching an image

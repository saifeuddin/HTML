## HTML <img> Tag loading Attribute


The loading attribute specifies whether a browser should load an image immediately or to defer loading of off-screen images until for example the user scrolls near them. attribute.

Tip:
Add loading="lazy" only to images which are positioned below the fold.




Syntax

        
    <img src="URL" loading="eager | lazy">

Attributes Values

  • eager

    Default. Loads an image immediately

  • lazy

    Defer loading of images until some conditions are met. 

        
        
        <img src="w3html.gif" alt="W3Schools.com" ismap loading="lazy">

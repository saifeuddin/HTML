## HTML <img> Tag referrerpolicy Attribute


$$ -- The referrerpolicy attribute specifies which referrer information to use when fetching an image.


Syntax

        
        
    <img referrerpolicy=" no-referrer| no-referrer-when-downgrade | 
        origin|origin-when-cross-origin| unsafe-url">

Attributes Values

        
        
    <img src="https://example.com/images/myimage.jpg" alt="Some image" referrerpolicy="no-referrer">

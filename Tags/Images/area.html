## HTML <area> Tag


$$ -- The area tag defines an area inside an image map (an image map is an image with clickable areas).

<area> elements are always nested inside a map tag.


Note : The usemap attribute in <img> is associated with the map element's name attribute, and creates a relationship between the image and the map.



All Attributes

  1. shape - default / rect / circle / poly

    Specifies the shape of the area


  2. coords - coordinates

    Specifies the coordinates of the area.


  3. alt - text

    Specifies an alternate text for the area. Required if the href attribute is present

    The alt attribute specifies an alternate text for an area, if the image cannot be displayed.

    The alt attribute is required if the href attribute is present. 
                    
    <area shape="circle" coords="124,58,8" href="venus.htm" alt="Venus">
  4. href - URL

    Specifies the hyperlink target for the area 
                    
    <area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm" >

  5. hreflang - HTML language_code

    Specifies the language of the target URL 
                    
    <area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm" hreflang="en">

  6. target - _blank / _self / _parent / _top / frameName

    Specifies where to open the target URL 
                    
    <area shape="rect" coords="0,0,82,126" href="sun.htm" alt="Sun" target="_blank">

  7. download - fileName

    Specifies that the target will be downloaded when a user clicks on the hyperlink. 
                    
    <area shape="rect" coords="0,0,82,126" href="sun.htm" download= "sun" alt="Sun" target="_blank"  >
  8. media

    Specifies what media/device the target URL is optimized for.

    Only used if the href attribute is present.


  9. type = media_type

    Specifies the media type of the target URL.

    This attribute is only used if the href attribute is set. 
                    
    <area shape="rect" coords="0,0,82,126" alt="Sun" href="/images/sun.gif" type="image/gif">
  10. rel

    Specifies the relationship between the current document and the target URL



  11. referrerpolicy

    Specifies which referrer information to send with the link.

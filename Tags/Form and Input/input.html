## HTML <form> Tag

Tips and Notes

Attribute :

  1. accept = values

    Specifies a filter for what file types the user can pick from the file input dialog box (only for type="file").


  2. autofocus

    Specifies that an input element should automatically get focus when the page loads.


  3. autocomplete = on / off

    Specifies whether a input element should have autocomplete on or off.


  4. alt = text

    Specifies an alternate text for images (only for type="image").


  5. checked =

    Specifies that an input element should be pre-selected when the page loads (for type="checkbox" or type="radio").


  6. disabled =

    Specifies that an input element should be disabled.


  7. dirname = input_name.dir

    Specifies that the text direction will be submitted.


  8. form = form_id

    Specifies the form the input element belongs to.


  9. formaction = URL

    Specifies the URL of the file that will process the input control when the form is submitted (for type="submit" and type="image").


  10. formmethod = GET / POST

    Defines the HTTP method for sending data to the action URL (for type="submit" and type="image")


  11. formenctype = values

    Specifies how the form-data should be encoded when submitting it to the server (for type="submit" and type="image").


  12. formnovalidate

    Specifies that the input should not be validated when submitted.


  13. formtarget = _black / _self / _top / _parent / frameName

    Specifies where to display the response that is received after submitting the form (for type="submit" and type="image").


  14. list = datalist_id

    Refers to a datalist element that contains pre-defined options for an input element.


  15. size = number

    Specifies the width, in characters, of an input element.

    Note: The size attribute works with the following input types: text, search, tel, url, email, and password.


  16. hight = pixels

    Specifies the hight of an input element (only for type="image").


  17. width = pixels

    Specifies the width of an input element (only for type="image").


  18. max = number

    Specifies the maximum value for an input element.


  19. maxlength = number

    Specifies the maximum number of characters allowed in an input element.


  20. min = number

    Specifies a minimum value for an input element.


  21. minlength = number

    Specifies the minimum number of characters allowed in an input element.


  22. multiple =

    Specifies that a user can enter more than one value in an input element ( only for type= 'file' and type= 'email')

    type="file": To select multiple files, hold down the CTRL or SHIFT key while selecting.

    input type="email" : Separate each email with a comma, like: mail@example.com, mail2@example.com, mail3@example.com in the email field.


  23. name = text

    Specifies the name of an input element.


  24. placeholder = text

    Specifies a short hint that describes the expected value of an input element

    Note: The placeholder attribute works with the following input types: text, search, url, tel, email, and password.


  25. pattern = regexp

    Specifies a regular expression that an input element's value is checked against.


  26. readonly

    Specifies that an input field is read-only.


  27. required

    Specifies that an input field must be filled out before submitting the form.

    Note: The required attribute works with the following input types: text, search, url, tel, email, password, date pickers, number, checkbox, radio, and file.


  28. src = URL

    Specifies the URL of the image to use as a submit button (only for type="image").


  29. step = values

    Specifies the interval between legal numbers in an input field


  30. value = text

    Specifies the value of an input element.

    • For "button", "reset", and "submit" - it defines the text on the button
    • For "text", "password", and "hidden" - it defines the initial (default) value of the input field
    • For "checkbox", "radio", "image" - it defines the value associated with the input (this is also the value that is sent on submit)
    Note: The value attribute cannot be used with input type="file"


  31. type = values

    Specifies the type input element to display.

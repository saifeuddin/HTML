## HTML <label> Tag


$$ -- The label tag defines a label for several elements:

Proper use of labels with the elements above will benefit:

  • Screen reader users (will read out loud the label, when the user is focused on the element)
  • Users who have difficulty clicking on very small regions (such as checkboxes) - because when a user clicks the text within the label element, it toggles the input (this increases the hit area).




Tip: The for attribute of label must be equal to the id attribute of the related element to bind them together.

A <label> can also be bound to an element by placing the element inside the <label> element.


Attributes

  1. for = element_idName

    Specifies the id of the form element the label should be bound to 

                    
        <label for="browser" id="form2"> Choose your browser from the list:</label>
  2. form = form_idName

    Specifies which form the label belongs to

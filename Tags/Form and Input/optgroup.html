## HTML <optgroup> Tag


$$ -- The optgroup tag is used to group related options in a select element (drop-down list).

If you have a long list of options, groups of related options are easier to handle for a user.



Attributes

  1. label = text

    Specifies a label for an option-group

  2. disabled

    Specifies that a optgroup should be disabled

    The disabled attribute can be set to keep a user from selecting the option-group until some other condition has been met (like selecting a checkbox, etc.).
    Then, a JavaScript is required to remove the disabled value, and make the option-group selectable.

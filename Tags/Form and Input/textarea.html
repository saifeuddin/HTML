## HTML <textarea> Tag


$$ -- defines a multi-line text input control.

textarea element is often used in a form, to collect user inputs like comments or reviews.



A text area can hold an unlimited number of characters, and the text renders in a fixed-width font (usually Courier).

The size of a text area is specified by the cols and rows attributes (or with CSS).

The name attribute is needed to reference the form data after the form is submitted (if you omit the name attribute, no data from the text area will be submitted).

The id attribute is needed to associate the text area with a label.



Attributes

  1. name = text

    The name attribute specifies a name for a text area.

    The name attribute is used to reference elements in a JavaScript, or to reference form data after a form is submitted. 
                        
          <textarea name="comment"> Enter text here...</textarea>
  2. dirname = textareaName.dir

    Specifies that the text direction of the textarea will be submitted.

    The dirname attribute's value is always the name of the text area, followed by ".dir". 
                        
          <textarea name="comment" dirname="comment"> Enter text here...</textarea>
  3. cols = number

    Specifies the visible width of a text area

  4. rows = number

    Specifies the visible number of lines in a text area

  5. form = form_id

    Specifies which form the text area belongs to

  6. autofocus =

    Specifies that a text area should automatically get focus when the page loads

  7. disabled =

    Specifies that a text area should be disabled

  8. readonly =

    Specifies that a text area should be read-only

  9. required =

    Specifies that a text area is required/must be filled out

  10. maxlength = number

    Specifies the maximum number of characters allowed in the text area

  11. wrap = soft / hard

    Specifies how the text in a text area is to be wrapped when submitted in a form

  12. placeholder = text

    Specifies a short hint that describes the expected value of a text area

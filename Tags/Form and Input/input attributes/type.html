## HTML <input> Tag type Attribute


$$ -- The type attribute specifies the type of input element to display.

Syntax 

        
        input type="value"

Attributes Values

  1. text

    Default. Defines a single-line text field

    The default width of the text field is 20 characters. 
                        
    <input type="text" id="fname" name="fname">
  2. number

    Defines a field for entering a number 

                        
    <input type="number" id="quantity" name="quantity" min="1" max="5">
  3. tel

    Defines a field for entering a telephone number 

                        
    <input type="tel" id="phone" name="phone" placeholder="123-45-678" pattern="[0-9]{3}-[0-9]{2}-[0-9]{3}" required>
  4. password

    Defines a password field

    Note: Any forms involving sensitive information like passwords should be served over HTTPS. 
                        
    <input type="password" id="pwd" name="pwd">
  5. submit

    defines a submit button which submits all form values to a form-handler.

    The form-handler is typically a server page with a script for processing the input data. 
                        
    <input type="submit"  value="Submit">
  6. button

    Defines a clickable button (mostly used with a JavaScript to activate a script) 

                    
    <input type="button" value="Click me" onclick="msg()">
  7. reset

    Defines a clickable button (mostly used with a JavaScript to activate a script)

    Tip: Avoid reset buttons in your forms! It is frustrating for users if they click them by mistake. 
                    
    <input type="reset"">
  8. range

    Defines a range control (like a slider control) 

                    
    <input type="range" id="points" name="points" min="0" max="10">
  9. checkbox

    Defines a checkbox 

                    
    <input type="checkbox" id="vehicle1" name="vehicle1" value="Bike">
  10. radio

    Defines a radio button

    Note: The value attribute defines the unique value associated with each radio button. The value is not shown to the user, but is the value that is sent to the server on "submit" to identify which radio button that was selected.


    Note: The radio group must have share the same name (the value of the name attribute) to be treated as a group. Once the radio group is created, selecting any radio button in that group automatically deselects any other selected radio button in the same group. You can have as many radio groups on a page as you want, as long as each group has its own name. 
                    
    <input type="radio" id="html" name="fav_language" value="HTML">
  11. color

    Defines a color picker

    The default value is #000000 (black). The value must be in seven-character hexadecimal notation. 
                        
    <input type="color" id="favcolor" name="favcolor" value="#ff0000">
  12. date

    Defines a date control (year, month, day (no time)) 

                        
    <input type="date" id="birthday" name="birthday">
  13. datetime-local

    Defines a date and time control (year, month, day, time (no timezone) 

                        
    <input type="datetime-local" id="birthdaytime" name="birthdaytime">
  14. month

    Defines a month and year control (no timezone)

    The format is "YYYY-MM". 
                        
    <input type="month" id="bdaymonth" name="bdaymonth">
  15. week

    Defines a week and year control (no timezone) 

                        
    <input type="week" id="week" name="week">
  16. time

    Defines a control for entering a time (no timezone) 

                        
    <input type="time" id="appt" name="appt">
  17. email

    Defines a field for an e-mail address 

                        
    <input type="email" id="email" name="email">
  18. file

    Defines a file-select field and a "Browse" button (for file uploads) 

                        
    <input type="file" id="myfile" name="myfile">
  19. image

    Defines an image as the submit button 

                        
    <input type="image" alt="submit" src="img_submit.gif"width="48" height="48">
  20. hidden

    Defines a hidden input field 

                        
    <input type="hidden" id="custId" name="custId" value="3487">
  21. url

    Defines a field for entering a URL 

                        
    <input type="url" id="homepage" name="homepage">
  22. search

    Defines a text field for entering a search string

    Note: Remember to set a name for the search field, otherwise nothing will be submitted. The most common name for search inputs is q. 
                        
    <input type="search" id="gsearch" name="gsearch">

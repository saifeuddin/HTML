## HTML onabort Attribute








HTML Audio/Video DOM onabort Event

Syntax

        
        
            HTML
        <video onabort= alert("Video load Aborted")>
        



            javaScript
        var vid = document.getElementById('');
        vid.onabort= function() {
            alert("Video alert aborted");
        }
        

            javaScript
        var vid = document.getElementById('').addEventListener("abort", myF);
        function myF() {
            alert('Hello!');
        }
